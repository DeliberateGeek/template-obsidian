# Decision Rules

Guidelines and conventions specific to this vault.

## Folder Organization

| Folder | When to use |
|--------|------------|
| `📥 Inbox/` | Quick captures, unsorted notes — process regularly |
| `📦 Archive/` | Completed, retired, or superseded content |
| `🫥 Attachments/` | Images, PDFs, and other media files |
| `🫥 Templates/` | Templater templates for new notes |
